Put your auth page photos in this folder.

Required file names:
- login-side.jpg      (used on /login, left side panel)
- register-side.jpg   (used on /register, right side panel)

Notes:

- Minimum recommended size: 900x1200
- If you only have one photo, copy it twice and rename to both file names above.
